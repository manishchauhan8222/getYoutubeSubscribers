Almabetter BackEnd Project

Project Name - (Get Youtube Subscribers)

Note - Try the routes given below by entering into the url path of the project

  • GET /subscribers : Returns an array of all subscribers details in the database.
  • GET /subscribers/names : Returns an array of all subscribers with only two feilds, their names and subscribed channel.
  • GET /subscribers/:id : Returns the details of subscriber with the given ID.
